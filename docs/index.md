# Take Word 文档

本文档面向 Take Word 的用户、分发者和开源仓库访问者，介绍安装、词库准备、手写识别、学习流程、复习模式、设置和隐私授权信息。

## 文档目录

- [开始使用](getting-started.md)：安装应用并完成第一次练习。
- [学习流程](learning-flow.md)：了解组内多阶段学习方式。
- [手写识别](handwriting-recognition.md)：使用画布、识别反馈和撤销清空逻辑。
- [词库格式](word-bank.md)：准备可导入的 CSV 词库。
- [复习与进度](review-and-progress.md)：理解复习入口和进度数据。
- [设置说明](settings.md)：配置学习强度、识别、词库和外观。
- [隐私与授权](privacy-and-licenses.md)：了解本地数据、第三方组件和素材责任。
- [常见问题](faq.md)：快速解决常见疑问。

## 推荐阅读顺序

首次使用建议依次阅读：

1. [开始使用](getting-started.md)
2. [词库格式](word-bank.md)
3. [学习流程](learning-flow.md)

已有词库的用户可直接阅读 [词库格式](word-bank.md)。
