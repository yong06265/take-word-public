# 设置说明

Take Word 设置页分为三个主要类别：进度与数据统计、词库、外观。

## 进度与数据统计

该类别用于调整学习行为和查看学习数据。

常见项目包括：

- 每组单词数量。
- 学习强度。
- 流程开关。
- 进度数据管理。

学习强度会影响一个单词在本组中出现的次数。轻量模式减少重复，标准模式保持平衡，强化模式会增加难词和错词的练习次数。

## 词库

词库类别用于：

- 选择内置词库。
- 导入 CSV 词库。
- 查看导入错误行。
- 使用示例词库。

词库管理集中放在设置页中，学习页面不会常驻显示导入入口，以保持画布专注。

## 外观

外观类别用于调整视觉和书写体验。

手写相关选项包括：

- 笔刷预设。
- 压感灵敏度。
- 压力曲线。
- 个性化手写识别。
- 识别详情显示。

默认界面方向为浅色、极简、清晰，并使用轻微液态玻璃质感和柔和动效。

## 发音

新词出现时可以自动朗读，并支持重复朗读，形成接近听写的节奏。

错误后重新尝试时通常只朗读一次，避免干扰用户重新默写。

词条包含音频时优先播放词条音频，无法播放时使用 Android TextToSpeech 兜底。

## 识别详情

识别详情适合调试或高级用户使用。开启后可以查看候选结果、错误标签和相似度等信息。普通学习中建议保持简洁反馈。
